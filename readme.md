## Quickstart
<!-- * install [sMQTTBroker library](https://github.com/terrorsl/sMQTTBroker)
  (you can use the Arduino library manager and search for sMQTTBroker)
* ~~make sMQTTConfig.h~~ -->

```c++
#include<sMQTTBroker.h>
```
```c++
sMQTTBroker broker;
#define PORT 1883
```
```c++
void setup(){
  broker.init(PORT);
};
```
```c++
void loop(){
  broker.update();
};
```
